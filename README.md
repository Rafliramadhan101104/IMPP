# IMPP (Produksi Problem)

## Deskripsi
IMPP (Produksi Problem) adalah aplikasi berbasis Laravel yang digunakan untuk membantu pengelolaan data permasalahan produksi. Sistem ini memudahkan pengguna dalam mencatat, mengelola, dan memantau data produksi secara terstruktur.

## Fitur
- Login
- Dashboard
- Kelola Data Produksi
- Kelola Lokasi Line
- Kelola User
- CRUD Data
- Database MySQL

## Teknologi
- Laravel
- PHP
- MySQL
- Bootstrap
- HTML, CSS, JavaScript

## Cara Menjalankan

```bash
git clone https://github.com/Rafliramadhan101104/IMPP.git
```

```bash
cd IMPP
```

```bash
composer install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

Atur konfigurasi database pada file `.env`, kemudian jalankan:

```bash
php artisan migrate
```

Terakhir jalankan server:

```bash
php artisan serve
```
